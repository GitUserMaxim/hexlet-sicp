<p>Следующая процедура вычисляет математическую функцию, называемую функцией Аккермана.</p>
<pre><code>(define (A x y)
  (cond ((= y 0) 0)
        ((= x 0) (* 2 y))
        ((= y 1) 2)
        (else (A (- x 1)
                 (A x (- y 1))))))
</code></pre>
<p>Каковы значения следующих выражений?</p>
<pre><code>(A 1 10)

(A 2 4)

(A 3 3)
</code></pre>
<p>Рассмотрим следующие процедуры, где 
<code>A</code>
 — процедура, определенная выше:
</p>
<pre><code>(define (f n) (A 0 n))

(define (g n) (A 1 n))

(define (h n) (A 2 n))

(define (k n) (* 5 n n))
</code></pre>
<p>Дайте краткие математические определения функций, вычисляемых процедурами 
<code>f, g</code>
 и 
<code>h</code>
 для положительных целых значений 
<code>n</code>
. Например, 
<code>(k n)</code>
 вычисляет
<code>5 * n * n</code>
.
